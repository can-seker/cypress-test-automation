## Install 

# Node.js Install
Run this commands: 
1. **curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash -**
2. **sudo apt-get install -y nodejs**

# Cypress Install
Run this command: **npm install cypress**

---

## Run tests
Run this command: **npx cypress run**

## Run Dashboard
Run this command: **npx cypress open**
